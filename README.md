# Co-op Gamified Task Tracker (Backend API)

> **Status: On-going Development**

## Project Idea

This is a gamified task management platform (To-do list) designed to break the monotony of traditional task boards. 

Users and their teams commit to shared goals, and each group owns a "virtual pet." The condition of this pet depends entirely on the team's progress:
- **Evolve and Grow:** Every time a member completes a task on time, the pet gains experience points (EXP) and its happiness index increases.
- **Penalties for Slacking:** If any member misses a deadline, the system automatically deducts the pet's health points (HP) and mood index.

This project applies behavioral psychology to create motivation and boost productivity through collective responsibility and a highly visual reward system.
